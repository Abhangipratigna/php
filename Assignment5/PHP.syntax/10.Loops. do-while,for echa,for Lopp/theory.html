Ans.10 Discuss the difference between for loop, foreach loop, and do-while loop in PHP.

    for loop

  • The iteration is clearly visible. The block of code is repeated as long as the condition is met or the counter meets a specific value.
  • Good performance.
  • The stop condition is specified easily.
  • upon working with collections, it needs the usage of the count() function.

    • foreach loop

  • The iteration is hidden. The block of code is repeated until iterating over the array is completed.
  • Better performance.
  • The stop condition has to be explicitly specified.
  • it can simply work without the usage of the count() method.

    • do-while loop

  • In a while loop, the condition is checked before the loop body executes. So the body may not
    execute even once if the condition is false from the beginning. On the other hand,
    in a do while loop the body is guaranteed to execute at least once because the
    condition is checked after executing the body.
